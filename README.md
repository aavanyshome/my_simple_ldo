The PMOS folded cascode based LDO uses a folded cascode error amplifier to improve the performance of the voltage regulator. The circuit mainly consists of a PMOS differential input pair, cascode transistors, current mirrors, a pass transistor, and a feedback network. The PMOS input transistors compare the reference voltage with the feedback voltage taken from the output. The folded cascode structure increases the gain and output resistance of the amplifier, which helps the regulator achieve better voltage regulation and improved stability. PMOS input devices are also used to reduce flicker noise and improve low-noise operation.
The working of the circuit is based on negative feedback. When the output voltage changes due to variations in input supply or load current, the feedback network senses this change and sends it to the error amplifier. The folded cascode amplifier amplifies the error signal and controls the PMOS pass transistor accordingly. If the output voltage decreases, the pass transistor allows more current to flow to the load, and if the output voltage increases, the current flow is reduced. In this way, the LDO maintains a constant regulated output voltage.
By using the folded cascode architecture, the LDO achieves higher loop gain, improved bandwidth, and better frequency response compared to the single-stage design. The regulator shows improved PSRR of about 70 dB at low frequency, reduced output noise, and stable operation even at higher load currents. The design also maintains a low dropout voltage of less than 200 mV while supporting up to 100 mA load current with moderate power consumption.
